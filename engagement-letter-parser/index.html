1What this is

A prompt template that turns any modern AI into an engagement-letter parser. Drop in the PDF, get every field you need to set up the order — formatted cleanly for typing into the TOTAL Assignment Tab or any other appraisal software.

Fields it extracts:

Property address
Form type (1004 / 1004D / 1073 / 2055 / etc.)
Borrower (name, phone, email)
Lender / Client (company name)
Lender mailing address (street, city, state, zip)
AMC, if different from the lender
Fee
Due date + date ordered
Loan type (Conv / FHA / VA / USDA)
Loan purpose (Purchase / Refi / Construction)
Sale price (purchase orders)
Loan #, FHA case #, VA case #
Legal description
Special instructions
Borrower POC, Lender POC, Buyer's Agent, Seller's Agent, Tidewater POC
(machine-readable JSON output too, for spreadsheet imports)

Engagement letters from different AMCs (Mercury VMP, Aloft, ClassValuation, UWM, Reggora, etc.) all use different layouts. The prompt extracts by meaning, not by literal label — so the same prompt works regardless of which AMC sent the order.

2How to use it

  1. Click Copy prompt to clipboard above.
  2. Open Claude.ai or ChatGPT in another tab.
  3. Start a new conversation. Paste the prompt.
  4. Attach the engagement letter PDF using the paperclip / attachment icon.
  5. Send. Wait 20–40 seconds.
  6. Copy the extracted fields straight into TOTAL's Assignment Tab, or use the JSON for an import script.

Tip: most orders have only ONE of loan number / FHA case # / VA case #. The prompt is told not to copy the same number into multiple fields.

3Which AI to use

Any modern AI that supports PDF attachments. You don't need vision for this one — engagement letters are text-based.

Claude.ai (Pro plan)

Recommended. Sonnet is fast enough and accurate; Opus for the trickiest layouts. About $20/month.

ChatGPT (Plus plan)

Also works. Use the current top model. About $20/month.

Free-tier AIs typically don't support PDF attachments. Stick with paid plans.
