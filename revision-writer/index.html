1What this is

A prompt template that turns any modern AI into a revision-response drafter. Give it your original appraisal report plus the revision request (PDF, email screenshot, or pasted text), and it produces addendum-ready response text in the appraiser's voice — ready to paste into the report's addendum section.

Each item in the revision request is categorized before any drafting happens:

Already addressed — answer is in the report; cites the exact location
Legitimate correction — real error or inconsistency; will be revised
Reasonable clarification — adds language to make existing intent explicit
Misunderstanding — client misread something; courteous correction
Out of scope / unsupportable — declines professionally with USPAP / standard citation

Output is third-person report voice, no email salutations, no "thank you for your review" — paste it straight into TOTAL's addendum and deliver.

2How to use it

  1. Click Copy prompt to clipboard above.
  2. Open Claude.ai or ChatGPT in another tab.
  3. Start a new conversation. Paste the prompt into the message box (don't send yet).
  4. Attach both documents:
    • The original appraisal report PDF.
    • The revision request — PDF, email export, AMC portal screenshot, or pasted text (if pasted, drop it directly into the message under the prompt).
  5. Send. Wait 30–60 seconds.
  6. Review the draft, fill any [PLACEHOLDER] notes, paste into your report's addendum.

Tip: the prompt explicitly tells the AI to decline unsupportable requests (predetermined value, removal of a supported adjustment, etc.) and to cite USPAP appraiser independence. If a revision item pressures a value outcome, the response will say so plainly — that's a feature, not a bug.

4What the output looks like

The draft is ready to paste into the report. Format:

On [DATE] a revision request was received from the client. The request and response are as follows:

1. The client requests an explanation for the +$30,000 basement adjustment applied to Comparable 1.

   A: The basement adjustment reflects the contributory value of the subject's 1,500 sf unfinished basement relative to Comparable 1, which has no basement. See Sales Comparison Approach grid, basement line, and the Sales Comparison commentary on page 2. The adjustment was developed using the depreciated cost method, with the cost-new figure taken from the Cost Approach in this report and depreciation applied per the age-life method.

2. The client requests removal of Comparable 3 as a "non-comp" given its smaller GLA.

   A: Comparable 3 was selected for inclusion because it is the closest recent sale in the subject's immediate neighborhood within the relevant date range. GLA differences are addressed through a supported adjustment, and Comparable 3 received reduced reconciliation weight as documented in the Sales Comparison reconciliation. Removal of an otherwise valid comparable solely on size, without a substitute of equal or greater similarity, would compromise the supportability of the value opinion. See USPAP Standards Rule 1-1 and Standards Rule 2-2.

Nothing further requires comment in response to this request.

---
Items for review:
- Item 2 declines a removal request. The appraiser may want to soften phrasing if the client relationship warrants it.

Any spot where the AI needed information not present in either document shows up as a [PLACEHOLDER] note — fill those in before pasting into TOTAL.
